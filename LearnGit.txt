git branch XXXX
本地创建分支
git push origin XXXX
将新分支推送到服务器

====================================

git branch -d xxxxx
删除分支；
不能删除当前自己机所在分之；
git push origin :xxxx
删除远程仓库；

git branch -D xxxxx
测试也只是删除了本地分支；
git push origin :xxxx
删除远程仓库；

====================================

git tag
显示所有标签
git tag v1.0
打标签
git push origin v1.0
推送指定标签
git tag -d v1.0
删除本地标签
git push origin :refs/tags/v1.0
删除远程标签
